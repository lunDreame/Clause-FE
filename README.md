# Clause (클로즈)

PDF 계약서를 업로드하면 확인이 필요한 조항을 정리하고, 주의 포인트와 협상 문구를 제공하는 웹 애플리케이션입니다.

## 주요 기능

- PDF 계약서 업로드: 드래그 앤 드롭 또는 클릭으로 간편하게 업로드
- 자동 조항 분석: AI 기반으로 계약서의 주요 조항을 분석
- 리스크 레벨 분류: 주의 필요 / 확인 권장 / 양호로 분류
- 조항 핀 기능: 중요한 조항을 저장하여 나중에 확인
- 협상 문구 제공: 각 조항에 대한 협상 문구 제안
- 분석 요약: 차트와 통계로 한눈에 파악
- JSON 내보내기: 분석 결과를 JSON 파일로 다운로드
- 반응형 디자인: 모바일과 데스크탑 모두 지원

## 면책 조항

Clause는 법률 자문이 아니며, 정보 제공 목적입니다. 중요한 계약은 반드시 전문가 상담을 받으시기 바랍니다.
